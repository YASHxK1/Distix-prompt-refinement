prompt-refinement

Settings

Choose an LLM provider and model. API keys stay in Chrome's local extension storage and are used only when you choose Refine or test the connection.

Use a provider/model identifier, for example deepseek/deepseek-v4-pro.

Data handling

  • The API key is never sent to supported websites or content scripts.
  • Only the prompt you explicitly refine is sent to the selected provider.
  • No prompt history, analytics, or accounts are included.
